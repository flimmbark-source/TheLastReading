# The Last Reading - Attic MVP Asset Pack

This pack contains placeholder-production assets for the first attic rummage MVP.

## Core loop these assets support

Finish full Reading -> earn Candlelight -> transition to attic -> spend Candlelight on three rummage props -> collect found items -> return to table.

## Required first-use mapping

- `props/newspaper_stack_closed.png` -> `props/newspaper_stack_moved.png` -> reveals `clipping_01`
- `props/covered_frame_closed.png` -> `props/covered_frame_uncovered.png` -> reveals `photo_01`
- `props/old_coat_closed.png` -> `props/old_coat_searched.png` -> reveals `letter_01`

## Recommended first placement

- Newspaper stack: bottom-left / floor area.
- Covered frame: right-center / wall or floor lean.
- Old coat: left-center / chair or coat hook.

## Notes

These assets are meant to prove the interaction loop before final illustration. They are transparent PNGs except for the portrait attic background and full-screen transition overlays.
