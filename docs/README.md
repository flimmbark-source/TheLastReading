# The Last Reading — Real Attic Assets

Copy the folders inside this package into the repository root so they replace the existing `backgrounds/`, `props/`, `ui/`, `fx/`, and `asset_manifest.json` files.

The attic background uses the same filename as the old placeholder so current code still resolves it, but the image is now a landscape desktop asset. On mobile, the scene should keep this native landscape size and allow horizontal panning/scrolling rather than fitting to portrait.
